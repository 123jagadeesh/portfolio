FarmCart - MERN-Based E-Commerce Platform

Project Overview

FarmCart is an e-commerce platform built using the MERN stack that connects farmers directly with consumers, eliminating intermediaries. It integrates data analytics to help farmers optimize their crop choices based on demand and supply trends.

Technologies Used

  • Frontend: React.js, HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Additional Tools: Redux, JWT Authentication, Cloudinary for image uploads

Features

  • Direct farmer-to-consumer marketplace
  • Crop recommendation system using data analytics
  • Secure payment gateway integration
  • User authentication and role-based access
