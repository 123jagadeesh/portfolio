LinkedIn Job Search Bot - AI/ML Automation

Project Overview

This project is an automated job search bot that scrapes LinkedIn job postings using Selenium and BeautifulSoup. It collects job details based on user-defined search parameters and saves structured data in JSON format.

Technologies Used

  • Programming Language: Python
  • Libraries: Selenium, BeautifulSoup, Pandas, OS, JSON
  • Automation: Web Scraping with Selenium WebDriver

Features

  • Automated job search based on title and location
  • Extracts job details like company name, job title, and application link
  • Exports data to structured JSON format for analysis
